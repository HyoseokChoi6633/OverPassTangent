DLL 프로젝트를 생성 후
View 프로젝트에서 DLL 연결법

1. 속성 => VC++ 디렉토리 => 일반의 [포함 디렉토리] 항목의 끝에 추가
;../Chaos_Trigonometric_test
[여기에선 생성된 DLL 프로젝트의 디렉토리 명이다.]
[Debug, Release 공용]
[여기는 .h 파일을 불러오기 위해]


2. 속성 => VC++ 디렉토리 => 일반의 [라이브러리 디렉터리] 항목의 끝에 추가
;../x64/Release
[Release 모드 일때]
;../x64/Debug
[Debug 모드 일때]
[.dll 파일을 연결하기 위해]


3. 속성 => 링커 => 입력 => [추가 종속성] 항목의 끝에 추가
;Chaos_Trigonometric_test.lib
[생성된 라이브러리를 추가 종속성에 입력해야 한다.]
[Debug, Release 공용]
[빌드시 .lib 파일을 연결하기 위해]


4. 빌드시 Dll 부터 빌드 시작하고
   View 프로그램을 마지막에 빌드 하고 실행하여야 한다.
